# Medicine ERP - Portable Package

This folder contains the portable version of Medicine ERP.

INSTRUCTIONS:
1. Copy this entire folder to your target PC
2. Keep all files and folders in the same location as the .exe file
3. Run 'Medicine_ERP.exe' to start the application
4. No Python installation required on the target PC!

FOLDER STRUCTURE:
- Medicine_ERP.exe     - Main application
- logo.ico             - Application icon
- customer_bills/      - Stores PDF bills
- data/                - Excel database files
  - customer_details.xlsx
  - store_details.xlsx
  - customer_bills.xlsx
  - distributor_details.xlsx
  - distributor_transactions.xlsx
  - credit_debit.xlsx


NOTE: 
- The first time you run, it may take a few seconds to initialize.
- Keep the folder structure intact for proper functioning.
- All data is stored in the Excel files within the data/ folder.
